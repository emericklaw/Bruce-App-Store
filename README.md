App store for Bruce devices with over-the-air install and updates for scripts and themes.

## Features

- **Wireless Installation** - Download scripts directly from GitHub repository
- **Version Management** - Track installed versions and detect updates
- **Update Detection** - Automatically identifies when updates are available
- **Status Indicators** - Clear visual feedback on installation status

## Controls

- **Next Button**: Navigate to next item (category or app/theme)
- **Previous Button**: Navigate to previous item (category or app/theme)
- **Select Button**: Select, install, update, reinstall or delete the currently selected item
- **Exit/ESC**: Go back or close the App Store

## Status Indicators

The installer shows different status indicators for each item:

- **NOT INSTALLED** (Yellow) - Not currently installed
- **UPDATE AVAILABLE** (Orange) - Newer version is available for download
- **UP TO DATE** (Green) - Latest version is already installed

## Usage

1. Ensure your Bruce device is connected to WiFi
2. Run the App Store script
3. Browse categories using Next/Previous buttons
4. Browse available apps/themes using Next/Previous buttons
5. Select app/theme to install or update with the Select button
6. Monitor installation progress and status messages
7. Exit when finished

## Installation

Place the `App Store.js` file in your Bruce device's `/scripts` directory and run it through the `JS Interpreter` menu.
